This construction cost estimate application provides a method of storing and maintaining unit costs as well as creating cost estimates with the data. All parameters are created by the user and can be customized to meet individual needs.


Instuctions

  1. Use Add Category and Add Location buttons to enter options to be used when entering new items.
  2. Use Add Item to enter new items individually.
  3. When an item is found other options (Change, Update, Remove and Add) can be used.
  4. Change allows everything except the unit cost to be changed.
  5. Update allows only the price to be changed.
  6. Remove deletes an item from the database.
  7. Add To Estimate will add item to an estimate(new or existing).
  8. Use Search bar to view and item only.
  9. Upload file format must be CSV.
